Visual Studio Code est un éditeur de code extensible développé par Microsoft pour Windows, Linux et macOS2.

Vim est un éditeur de texte, c’est-à-dire un logiciel permettant la manipulation de fichiers texte. Il est directement inspiré de vi (un éditeur très répandu sur les systèmes d’exploitation de type Unix).

PhpStorm est un éditeur pour PHP3, HTML, CSS et JavaScript4, édité par JetBrains. Il permet d'éditer du code PHP pour les versions allant de la 5.3 à la 7.45
